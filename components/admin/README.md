# Компоненты админ-панели

## История действий (AuditHistoryDashboard.tsx)

Компонент для отображения истории действий пользователей в админ-панели. Предоставляет таблицу с информацией о том, кто, что и когда делал в системе.

### Возможности:
- Отображение списка действий пользователей
- Фильтрация по типу объекта и действию
- Пагинация результатов
- Визуальное выделение типов действий (создание, редактирование, удаление)

### Использование:
```tsx
import AuditHistoryDashboard from '@/components/admin/AuditHistoryDashboard';

export default function AdminDashboard() {
  return (
    <div>
      <h1>Админ-панель</h1>
      <AuditHistoryDashboard />
    </div>
  );
}
```

### Зависимости:
- `@/utils/supabase/client` - клиент Supabase для взаимодействия с базой данных
- `@/utils/audit-service` - сервис аудита для получения данных
